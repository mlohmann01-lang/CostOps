# Mockup Sandbox

## Environment variables

- `PORT` (default `4173`): dev/preview server port.
- `BASE_PATH` (default `/`): base path used by Vite for preview route mounting.
- `VITE_ECONOMIC_OPERATIONS_API_BASE` (optional): base URL for Economic Operations APIs.

Build and typecheck work without manual environment export because defaults are set in `vite.config.ts`.
