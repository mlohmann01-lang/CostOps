# Flexera Authority Model

Flexera provides entitlement and license-position evidence. Certen governs action, verifies outcomes, and prevents drift.

## Role
- Entitlement authority evidence
- License position confidence input
- Overlap and mismatch validation
- Confidence boost / block reasons for recommendations

## Demo vs Live
- Demo mode uses synthetic entitlement signals and explicit labels.
- If not configured, UI states: Flexera not connected.
- No live enterprise mutation implied.
