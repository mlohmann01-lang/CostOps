# m365 workflow escalation

Workflow escalation traceability includes:
- created
- assigned
- escalated
- reassigned
- sla breached
- exception approved
- exception revoked
- resolved

Escalation trace fields:
- workflowId
- recommendationId
- tenantId
- assignedOperator
- priority
- slaTarget
- escalationReason
- resolutionReason
- timestamp
