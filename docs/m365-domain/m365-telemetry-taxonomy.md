# m365 telemetry taxonomy

Canonical event set for M365 Phase C:
- M365_EVIDENCE_NORMALIZED
- M365_EVIDENCE_UNKNOWN
- M365_TRUST_DEGRADED
- M365_RECONCILIATION_FINDING_CREATED
- M365_RECOMMENDATION_GENERATED
- M365_RECOMMENDATION_SUPPRESSED
- M365_LIFECYCLE_STATE_DERIVED
- M365_GOVERNANCE_ESCALATED
- M365_PLAYBOOK_CONFLICT_SUPPRESSED
- M365_WORKFLOW_REVIEW_CREATED
- M365_WORKFLOW_ESCALATED
- M365_SIMULATION_CREATED
- M365_SIMULATION_LIFECYCLE_GATED
- M365_RENEWAL_READINESS_AGGREGATED
- M365_OUTCOME_RESOLVED
- M365_OUTCOME_DRIFT_DETECTED
- M365_OUTCOME_REVERSED
- M365_REPLAY_VALIDATED
- M365_REPLAY_MISMATCH

Required metadata by event context:
- tenantId
- recommendationId
- workflowId
- playbookId
- entityId
- correlationId
- traceId
- lifecycleState
- trustBand
- governanceOutcome
- simulationId
- outcomeId
- severity
- eventTimestamp
