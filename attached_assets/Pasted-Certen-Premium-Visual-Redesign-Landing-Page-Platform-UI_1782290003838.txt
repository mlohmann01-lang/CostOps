## Objective

Upgrade Certen from a standard dark SaaS interface into a premium executive-grade Economic Control System.

The goal is not just colour polish. The product must visually feel like:

* an executive command center
* a finance-grade control system
* an evidence-backed value platform
* a governance and proof layer

Use the provided mockup as direction for layout, hierarchy, dashboard composition and premium feel.

Do not copy the Certen.io logo, icon, wave motif, typography or brand identity.

Create an original Certen visual system.

# Part 3 — State-Aware UI Rules

Do not break the existing runtime state enforcement.

The redesign must preserve:

```text
DEMO
LIVE_UNCONNECTED
LIVE_DISCOVERING
LIVE_OPERATIONAL
```

Rules:

## DEMO

Can show synthetic charts, sample values and sample findings.

Must clearly show:

```text
Synthetic evidence only.
No production systems connected.
Live execution disabled.
```

## LIVE_UNCONNECTED

Must show:

```text
No production systems connected.
Connect Microsoft 365 or another supported platform to begin discovery.
```

Must not show:

* synthetic findings
* synthetic values
* fake charts
* fake finance verification
* fake trust percentages

In LIVE_UNCONNECTED charts should show empty states, not demo data.

## LIVE_DISCOVERING

Can show discovered values only.

Approved / Executed / Verified / Protected must show Pending until evidence exists.

## LIVE_OPERATIONAL

Can show full operational data.
