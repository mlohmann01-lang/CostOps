Executive Command Center Visual Dashboard v2

## Objective

The landing page redesign is complete.

Do not redesign the landing page further.

Focus exclusively on transforming the Executive Command Center from a text-heavy interface into a visual executive operating system.

The current Command Center is still:

* too many text cards
* too little visual hierarchy
* almost no charts
* no dashboard storytelling
* weak distinction between value, risk, proof and action

The goal is to make executives understand the state of their technology estate within 10 seconds.

---

# Critical Rule

Do not break runtime-state enforcement.

All visual components must respect:

```text
DEMO
LIVE_UNCONNECTED
LIVE_DISCOVERING
LIVE_OPERATIONAL
```

No synthetic values may appear in:

```text
LIVE_UNCONNECTED
```

Charts must render empty states instead.

```text

No Data
```
